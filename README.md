silentor, a friendly fork of [silent](https://github.com/fritx/silent)
========
>Be a silentor,just focus on marking your words down!

特点
----
* 开箱即用
* markdown only ,让你专注文章本身
* 友好的移动端支持
* 灵活的样式扩展

[DEMO](http://jayin.github.io/silentor/)
