silentor
========
>Be a silentor,just focus on mark your words down!

开发缘由
===
第一次使用fritx同学的[silent](https://github.com/fritx/silent) 失败了，于是决定重复造下轮子
所以，silentor是受silent的启发，代码简单容读。

Features
===
* markdown only .让你专注文章本身
* 无需后台,按照指定的地方写markdown就行了
* Github CSS

```json
author = {
    "name" : "Jayin",
    "data" : "2014-10-14",
    "thanks":"@fritx"
}
```
